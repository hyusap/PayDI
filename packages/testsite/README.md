# PayDI Test Store

Demo storefront for testing `paydi-widget` end-to-end with very low prices.

## Live Links

- Main hosted app: https://paydi.vercel.app/
- Hosted example store: https://paydi-test.vercel.app/

## Run Locally

From repo root:

```bash
bun run --cwd packages/testsite dev
```

Local URL: `http://localhost:3001`
